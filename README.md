# fprjc
Feminist for Peace Rights and Justice Center
## Overview
This project is a complete rebuild of the official **FPRJC (Feminist for Peace Rights and Justice Centre)** website using **React.js**.  
The goal is to create a modern, fast, and secure platform that supports advocacy, resource sharing, and member engagement.

## Tech Stack
- **Front-End:** React.js + Tailwind CSS  
- **Backend:** Node.js  
- **Database:** MongoDB 
- **Version Control:** Git + GitHub  
- **Hosting:** cPanel (React build upload via `/build` folder)
